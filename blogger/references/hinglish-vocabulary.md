## Hinglish Sentence Patterns

### Pattern 1: English + Hindi Word
```
i am kinda free all day , cllg is shit af
```
- `kinda` = English
- `cllg` = English abbreviation
- `shit af` = English slang

### Pattern 2: Hindi Structure + English Words
```
bhut diff hai
```
- `bhut` = Hindi (very)
- `diff` = English
- `hai` = Hindi (is)

### Pattern 3: Mixed Grammar
```
da what's the result of merging only embedtokens
```
- `da` = Hindi address
- `what's the result` = English
- `merging only embedtokens` = English technical

### Pattern 4: Hindi Interjections
```
kya kare bhai
```
- `kya kare` = Hindi (what to do)
- `bhai` = Hindi (brother)

### Pattern 5: English + Hindi Tag
```
that's good innit
```
- `that's good` = English
- `innit` = English slang (isn't it)
