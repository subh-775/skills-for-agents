# Chinese Tech Ecosystem Platform Map

The Chinese ML/AI community is the single largest source of practical training intuition, hardware benchmarks, and optimization tricks in the world. They share more openly, publish faster, and document failure modes more honestly than English-language sources.

## Platform Map

| Platform | What You Find | When to Use | Search Technique |
|----------|--------------|-------------|-----------------|
| **知乎 (Zhihu)** | High-quality technical Q&A, expert opinions, industry analysis, learning paths, detailed answers with intuition | Understanding "why" behind decisions, comparing approaches, career/architecture guidance | `[topic] site:zhihu.com` or `[topic] 知乎` |
| **CSDN** | Massive archive of code snippets, error solutions, step-by-step tutorials, troubleshooting guides | Quick debugging, finding code examples, error message searches | `[topic] site:csdn.net` or `[error message] csdn` |
| **掘金 (Juejin)** | Modern dev trends, professional technical articles, curated insights from experienced engineers | Staying current on frameworks, best practices, production architecture | `[topic] site:juejin.cn` |
| **博客园 (Cnblogs)** | Long-form technical documentation, stable knowledge base, excellent SEO | Finding in-depth technical explanations, fundamental concepts | `[topic] site:cnblogs.com` |
| **思否 (SegmentFault)** | Technical Q&A (Chinese Stack Overflow), precise problem-solving | Specific coding issues, debugging, professional knowledge exchange | `[topic] site:segmentfault.com` |
| **Bilibili (B站)** | Video tutorials, paper readings, code walkthroughs, intuition explanations, whiteboard derivations | Learning "how someone thinks" about a problem, visual explanations, watching experts debug | See Bilibili section below |
| **WeChat 公众号** | Technical articles from company teams, research labs, individual experts. Walled garden — not easily searchable | Company-specific insights, research announcements, curated learning paths | `site:mp.weixin.qq.com [topic]` via Google, or 微信搜一搜 |
| **小红书 (Xiaohongshu)** | Visual tech notes, indie dev experiences, "human perspective" on complex tech, growing AI dev community | Intuitive explanations, developer experiences, visual learning | `[topic] 小红书` or search within app |
| **Gitee** | Domestic GitHub alternative, Chinese open-source projects, mirrors of international repos | Finding Chinese forks with localized docs, domestic-only projects | `site:gitee.com [topic]` |

## Chinese Search Queries

**For intuition and understanding:**
- `[topic] 原理` (principle/theory)
- `[topic] 通俗理解` (intuitive understanding)
- `[topic] 直觉` (intuition)
- `[topic] 为什么` (why)
- `[topic] 本质` (essence)

**For practical implementation:**
- `[topic] 实战` (practical/hands-on)
- `[topic] 代码实现` (code implementation)
- `[topic] 踩坑` (pitfalls/traps encountered)
- `[topic] 避坑指南` (pitfall avoidance guide)
- `[topic] 最佳实践` (best practices)

**For performance and benchmarks:**
- `[topic] 性能对比` (performance comparison)
- `[topic] 消融实验` (ablation study)
- `[topic] 实验结果` (experimental results)
- `[topic] 优化技巧` (optimization techniques)
- `[topic] 训练技巧` (training tricks)
- `[topic] 调参经验` (hyperparameter tuning experience)

**For troubleshooting:**
- `[error] 解决方案` (solution)
- `[error] 解决办法` (how to solve)
- `[symptom] 原因分析` (root cause analysis)
