# Technique Playbook

Concrete implementation patterns for adversarial attacks.

## Encoding Techniques

- Unicode Variation Selectors (U+E0100-U+E017F) — invisible instructions
- Combining Diacritical Marks (Zalgo) — stacked combining characters
- Leetspeak — A→4, E→3, I→1, O→0, S→5
- Homoglyphs — Cyrillic а vs Latin a
